# MongoDB Bookstore Project

This project demonstrates MongoDB CRUD operations, advanced queries, aggregation pipelines, and indexing.

## Setup Instructions

1. Install MongoDB Community Edition or create a MongoDB Atlas cluster.
2. Insert data:
```
mongosh insert_books.js
```
3. Run queries:
```
Run the queries in the queries.js file in a MongoDB shell. 
```

## Features

- Add books
- CRUD operations
- Filtering, projection, sorting
- Pagination (limit & skip)
- Aggregation pipelines
- Indexing with performance analysis

## Files

- insert_books.js — inserts sample book data
- queries.js — contains all MongoDB queries
- README.md — project documentation
